Turn 1 用户 ▸ show me main.rs。Cogito ∴ 回复 + 一个已完成 collapsed 的 tool（✓ read_file 12ms）。后面是 agent 的文本回答。代码块当前是 dim 行（markdown 渲染留给下一 sprint）。
Turn 2 (进行中) 第一个 tool ▾ write_file 已展开（args + 结果摘要均 dim）。第二个 tool ⠋ run_tests 正在运行（黄 spinner + 已用时间）。最下面 ∴ ⠋ 表示 cogito 在等 tool 结果回来后继续思考（首字未出，spinner 等待中）。

没有的东西 没有 tools pane、没有 borders、没有持久 status bar、没有 turn 计数。Ctrl-T 快捷键也废了。

1-9快速展开最近 N 个 tool
e / c展开 / 折叠当前消息所有 tool

Esc取消 selection / 收起 popup
/开 slash 命令 popup（v0.1 行为保留）
