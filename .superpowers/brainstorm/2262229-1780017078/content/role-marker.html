A1. 几何符号

蓝 ▸ = 你、绿 ◆ = agent。零文字，最克制。内容缩进 3 格自然形成视觉列。

▎who are you?

▎I am cogito, an embedded
▎agent runtime.
▎
▎▸ read_file  ok  12ms
▎▾ grep  ok  45ms
▎  args: {"pattern": "fn main"}

▎what's the weather?

▎I can't access weather APIs
▎directly.

A2. 左侧色条

每行左侧一条彩色细竖线（蓝/绿）。引用块风格，多行消息很自然连成一段，视觉锚点强但不喧宾夺主。

A3. 时间 + 模型名

把 "cogito" 换成实际 model_id（如 opus-4.7 / claude-sonnet-4-6）。带时间戳，更像 chat 工具的轻量 header。多模型时身份明确。

› who are you?

  I am cogito, an embedded
  agent runtime.

  ▸ read_file  ok  12ms
  ▾ grep  ok  45ms
    args: {"pattern": "fn main"}

› what's the weather?

  I can't access weather APIs
  directly.

A4. 不对称缩进

你的输入有 › 前缀、零缩进；agent 回复无标记、缩进 2 格。极致克制，靠位置即区分。tool 调用天然属于 agent 段落内。
