C1. Idle Header

顶部 1 dim 行写身份信息（项目名 · 模型 · 策略 · session 前 8 位）。底部干净，只有 input。Key hints 只在输入框 focus 时浮现。Turn 计数完全没了。

C2. Hush

完全隐藏所有 chrome。只剩 chat + input。要看身份信息：/info 或 Ctrl-S 临时弹出一个 overlay。Key hints 同样按需召出（? 键）。最激进，最 distraction-free。

C3. Ambient

启动时一个 banner ( ∴∴∴ + 名称 + 模型 + 可用工具) 作为第一帧，随着对话自然向上滚走 —— 像 zsh shell 的 motd。底部 input 提示符里 ghost-text 显示当前模型名（› opus-4.7），不抢戏但能扫到。

关键问题：你想要"任何时候一眼能看到当前模型 / 策略"（→ C1），还是"完全沉浸，需要时才召出"（→ C2），还是"启动有仪式感，运行时低调"（→ C3）？
