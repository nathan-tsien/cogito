# Sprint 8 · Async Jobs Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship `LocalJobManager` and wire H08's async dispatch path end-to-end, so a model can call an `ExecutionClass::AlwaysAsync` tool (`run_tests`), the turn pauses, the job runs as a background `tokio::task`, and on completion the turn resumes — with crash-safe recovery via the conversation event log.

**Architecture:** New concrete `LocalJobManager` in `cogito-jobs` (in-memory `HashMap<JobId, JobLifecycle>` plus `tokio::task` submission). Brain side adds one additive event variant (`EventPayload::JobSubmitted { call_id, job_id, tool_name }`), activates the dormant `InFlight::PausedOnJob` and `SessionCommand::JobCompleted` paths in `session_loop`, and connects H08's `DispatchOutcome::AsyncJob` branch through `TurnDeps`-injected JobManager and per-session completion sink. Resume coordinator drops the Sprint 3 "find latest unmatched `ToolUseRecorded`" narrowing and reads `call_id` directly from `JobSubmitted`. A separate fix for the latent `cancel-token-disconnect` lands in this sprint because cancel-while-paused depends on it.

**Tech Stack:** Rust 2024 (MSRV 1.85), `async-trait`, `tokio::sync::{mpsc, broadcast}`, `tokio::process::Command`, `tokio::task::{spawn, AbortHandle}`, `parking_lot::Mutex`, `serde_json`, `tracing`, `thiserror`. Tests via `cargo nextest`; chaos via existing `cogito-test-fixtures` harness + `MockJobManager`.

**Branch:** `feat/sprint-8-async-jobs` (created from `main` after current head).

**Spec:** `docs/superpowers/specs/2026-05-24-sprint-8-async-jobs-design.md`

```rust
//! Cross-impl `JobManager` contract tests. Every `JobManager` implementation
//! must call `run_contract_suite(constructor)` from its own `tests/` module.
//!
//! Two contracts are exercised (spec §10.2):
//! 1. `on_complete` after terminal -> fire sink immediately.
//! 2. `on_complete` before terminal -> store sink; fire on completion.

#![allow(clippy::unwrap_used, clippy::expect_used, clippy::panic)]

use async_trait::async_trait;
use cogito_protocol::job::{
    JobCompletionEvent, JobError, JobId, JobManager, JobOutcome, JobStatus,
};
use cogito_protocol::tool::ToolResult;
use std::sync::Arc;
use std::time::Duration;
use tokio::sync::mpsc;

/// Implementations supply a constructor + a test-side `register` + `complete`
/// hook. `MockJobManager` provides these directly; `LocalJobManager` provides
/// `submit(fut)` whose `fut` resolves to the desired terminal outcome.
#[async_trait]
pub trait JobManagerHarness: Send + Sync + 'static {
    type Manager: JobManager + 'static;
    fn new() -> Arc<Self>;
    fn manager(&self) -> Arc<Self::Manager>;
    /// Drive a job to a registered + still-running state. Returns the JobId.
    async fn arm(&self, outcome: JobOutcome) -> JobId;
    /// Trigger completion of a previously-armed job with the outcome it was
    /// armed with.
    async fn fire(&self, job_id: JobId);
}

pub async fn run_contract_suite<H: JobManagerHarness>() {
    contract_1_fires_immediately::<H>().await;
    contract_2_fires_after_completion::<H>().await;
    contract_status_and_result_lifecycle::<H>().await;
    contract_sink_dropped_is_silent::<H>().await;
}

async fn contract_1_fires_immediately<H: JobManagerHarness>() {
    let harness = H::new();
    let mgr = harness.manager();
    let job_id = harness
        .arm(JobOutcome::Success { result: ToolResult::text("ok") })
        .await;
    harness.fire(job_id).await;

    let (tx, mut rx) = mpsc::channel(1);
    mgr.on_complete(job_id, tx).await.expect("on_complete ok");
    let evt = tokio::time::timeout(Duration::from_millis(200), rx.recv())
        .await
        .expect("sink fired in time")
        .expect("sender not dropped");
    assert_eq!(evt.job_id, job_id);
    assert!(matches!(evt.outcome, JobOutcome::Success { .. }));
}

async fn contract_2_fires_after_completion<H: JobManagerHarness>() {
    let harness = H::new();
    let mgr = harness.manager();
    let job_id = harness.arm(JobOutcome::Cancelled).await;

    let (tx, mut rx) = mpsc::channel(1);
    mgr.on_complete(job_id, tx).await.expect("on_complete ok");
    assert!(rx.try_recv().is_err(), "no event before fire()");
    harness.fire(job_id).await;
    let evt = tokio::time::timeout(Duration::from_millis(200), rx.recv())
        .await
        .expect("sink fired after completion")
        .expect("sender not dropped");
    assert!(matches!(evt.outcome, JobOutcome::Cancelled));
}

async fn contract_status_and_result_lifecycle<H: JobManagerHarness>() {
    let harness = H::new();
    let mgr = harness.manager();
    let unknown = JobId::default();
    assert!(matches!(mgr.status(unknown).await, Err(JobError::UnknownJob(_))));

    let job_id = harness
        .arm(JobOutcome::Failed { message: "boom".into() })
        .await;
    assert!(matches!(mgr.status(job_id).await.unwrap(), JobStatus::Running));
    assert!(matches!(mgr.result(job_id).await, Err(JobError::BackendUnavailable(_))));
    harness.fire(job_id).await;
    assert!(matches!(mgr.status(job_id).await.unwrap(), JobStatus::Failed));
    assert!(matches!(mgr.result(job_id).await.unwrap(), JobOutcome::Failed { .. }));
}

async fn contract_sink_dropped_is_silent<H: JobManagerHarness>() {
    let harness = H::new();
    let mgr = harness.manager();
    let job_id = harness
        .arm(JobOutcome::Success { result: ToolResult::text("ok") })
        .await;
    let (tx, rx) = mpsc::channel(1);
    mgr.on_complete(job_id, tx).await.expect("on_complete ok");
    drop(rx);
    // Firing with a dropped rx must not panic.
    harness.fire(job_id).await;
}
```

```rust
use cogito_jobs::LocalJobManager;
use cogito_protocol::job::{JobId, JobOutcome};
use cogito_protocol::test_support::contract_job_manager::{
    run_contract_suite, JobManagerHarness,
};
use std::sync::Arc;
use tokio::sync::Mutex;

struct LocalHarness {
    mgr: Arc<LocalJobManager>,
    armed: Mutex<Vec<(JobId, tokio::sync::oneshot::Sender<JobOutcome>)>>,
}

#[async_trait]
impl JobManagerHarness for LocalHarness {
    type Manager = LocalJobManager;

    fn new() -> Arc<Self> {
        Arc::new(Self {
            mgr: LocalJobManager::new(),
            armed: Mutex::new(Vec::new()),
        })
    }

    fn manager(&self) -> Arc<Self::Manager> {
        Arc::clone(&self.mgr)
    }

    async fn arm(&self, outcome: JobOutcome) -> JobId {
        let (tx, rx) = tokio::sync::oneshot::channel();
        let job_id = self.mgr.submit(async move {
            // Wait for fire() to send the outcome.
            rx.await.unwrap_or(JobOutcome::Cancelled)
        });
        self.armed.lock().await.push((job_id, tx));
        job_id
    }

    async fn fire(&self, job_id: JobId) {
        let mut guard = self.armed.lock().await;
        if let Some(pos) = guard.iter().position(|(id, _)| *id == job_id) {
            let (_, tx) = guard.remove(pos);
            // Lookup what outcome the test wants from the manager's stored
            // arm record. Simplest: send the outcome the contract suite
            // armed with. For this harness we re-read the stored job state
            // — but the simplest correct path is to capture outcome inside
            // arm() and clone here. Restructure if needed during impl.
            let _ = tx;
        }
    }
}

#[tokio::test]
async fn local_job_manager_satisfies_contract() {
    run_contract_suite::<LocalHarness>().await;
}
```

Note: the harness above has a TODO — `fire()` doesn't carry the original outcome. Either restructure `arm()` to store the outcome alongside the sender (recommended), or have `arm()` immediately resolve the future without waiting. The cleaner shape:

```rust
async fn arm(&self, outcome: JobOutcome) -> JobId {
    let outcome_clone = outcome.clone();
    let (gate_tx, gate_rx) = tokio::sync::oneshot::channel::<()>();
    let job_id = self.mgr.submit(async move {
        let _ = gate_rx.await;
        outcome_clone
    });
    self.armed.lock().await.push((job_id, gate_tx));
    job_id
}

async fn fire(&self, job_id: JobId) {
    let mut guard = self.armed.lock().await;
    if let Some(pos) = guard.iter().position(|(id, _)| *id == job_id) {
        let (_, tx) = guard.remove(pos);
        let _ = tx.send(());
    }
}
```

```rust
//! In-memory `JobManager` implementation. Jobs run as `tokio::task`s; their
//! lifecycle is tracked in a `HashMap<JobId, JobLifecycle>` behind a
//! `parking_lot::Mutex`. See `docs/superpowers/specs/2026-05-24-sprint-8-async-jobs-design.md`
//! §6 for the design rationale.

use std::collections::HashMap;
use std::future::Future;
use std::sync::Arc;

use async_trait::async_trait;
use cogito_protocol::job::{
    JobCompletionEvent, JobError, JobId, JobManager, JobOutcome, JobStatus,
};
use parking_lot::Mutex;
use tokio::sync::mpsc;
use tokio::task::AbortHandle;

/// Local job manager. One instance per `Runtime`; cloned via `Arc`.
pub struct LocalJobManager {
    jobs: Mutex<HashMap<JobId, JobLifecycle>>,
}

struct JobLifecycle {
    status: JobStatus,
    outcome: Option<JobOutcome>,
    on_complete_sink: Option<mpsc::Sender<JobCompletionEvent>>,
    abort_handle: Option<AbortHandle>,
}

impl LocalJobManager {
    /// Construct an empty manager.
    #[must_use]
    pub fn new() -> Arc<Self> {
        Arc::new(Self {
            jobs: Mutex::new(HashMap::new()),
        })
    }

    /// Submit a future as an async job. Returns immediately with the new
    /// `JobId`; the future runs on the current Tokio runtime. When the
    /// future resolves, the outcome is stored and any registered
    /// `on_complete` sink fires.
    ///
    /// Not part of the `JobManager` trait: only async tool implementations
    /// submit jobs, and they hold an `Arc<LocalJobManager>` directly.
    pub fn submit<F>(self: &Arc<Self>, fut: F) -> JobId
    where
        F: Future<Output = JobOutcome> + Send + 'static,
    {
        let job_id = JobId::default();
        let this = Arc::clone(self);
        let handle = tokio::spawn(async move {
            let outcome = fut.await;
            this.complete_internal(job_id, outcome).await;
        });
        let abort = handle.abort_handle();
        self.jobs.lock().insert(
            job_id,
            JobLifecycle {
                status: JobStatus::Running,
                outcome: None,
                on_complete_sink: None,
                abort_handle: Some(abort),
            },
        );
        job_id
    }

    /// Internal: called from the spawned task when the job future resolves.
    /// Transitions to terminal status and fires the registered sink.
    async fn complete_internal(&self, job_id: JobId, outcome: JobOutcome) {
        let sink = {
            let mut jobs = self.jobs.lock();
            let Some(job) = jobs.get_mut(&job_id) else {
                tracing::warn!(%job_id, "complete_internal for unknown job; dropping");
                return;
            };
            job.status = match &outcome {
                JobOutcome::Success { .. } => JobStatus::Completed,
                JobOutcome::Failed { .. } => JobStatus::Failed,
                JobOutcome::Cancelled => JobStatus::Cancelled,
                _ => JobStatus::Failed,
            };
            job.outcome = Some(outcome.clone());
            job.abort_handle = None;
            job.on_complete_sink.take()
        };
        if let Some(sink) = sink {
            // Best-effort send; if the sink is dropped, silently swallow.
            let _ = sink
                .send(JobCompletionEvent { job_id, outcome })
                .await;
        }
    }
}

#[async_trait]
impl JobManager for LocalJobManager {
    async fn status(&self, job_id: JobId) -> Result<JobStatus, JobError> {
        self.jobs
            .lock()
            .get(&job_id)
            .map(|j| j.status)
            .ok_or(JobError::UnknownJob(job_id))
    }

    async fn result(&self, job_id: JobId) -> Result<JobOutcome, JobError> {
        let jobs = self.jobs.lock();
        let Some(job) = jobs.get(&job_id) else {
            return Err(JobError::UnknownJob(job_id));
        };
        job.outcome.clone().ok_or_else(|| {
            JobError::BackendUnavailable(format!("job {job_id} has not yet completed"))
        })
    }

    async fn cancel(&self, _job_id: JobId) -> Result<(), JobError> {
        // Cancel is implemented in Task 4 of the Sprint 8 plan.
        Err(JobError::BackendUnavailable("cancel not yet implemented".into()))
    }

    async fn on_complete(
        &self,
        job_id: JobId,
        sink: mpsc::Sender<JobCompletionEvent>,
    ) -> Result<(), JobError> {
        let already_terminal = {
            let mut jobs = self.jobs.lock();
            let Some(job) = jobs.get_mut(&job_id) else {
                return Err(JobError::UnknownJob(job_id));
            };
            let is_terminal = matches!(
                job.status,
                JobStatus::Completed | JobStatus::Failed | JobStatus::Cancelled
            );
            if is_terminal {
                job.outcome.clone()
            } else {
                job.on_complete_sink = Some(sink.clone());
                None
            }
        };
        if let Some(outcome) = already_terminal {
            let _ = sink.send(JobCompletionEvent { job_id, outcome }).await;
        }
        Ok(())
    }
}
```

```rust
async fn cancel(&self, job_id: JobId) -> Result<(), JobError> {
    let (abort, sink) = {
        let mut jobs = self.jobs.lock();
        let Some(job) = jobs.get_mut(&job_id) else {
            return Err(JobError::UnknownJob(job_id));
        };
        // Already terminal -> no-op.
        if matches!(
            job.status,
            JobStatus::Completed | JobStatus::Failed | JobStatus::Cancelled
        ) {
            return Ok(());
        }
        job.status = JobStatus::Cancelled;
        let outcome = JobOutcome::Cancelled;
        job.outcome = Some(outcome.clone());
        let abort = job.abort_handle.take();
        let sink = job.on_complete_sink.take();
        (abort, sink)
    };
    if let Some(abort) = abort {
        abort.abort();
    }
    if let Some(sink) = sink {
        let _ = sink
            .send(JobCompletionEvent {
                job_id,
                outcome: JobOutcome::Cancelled,
            })
            .await;
    }
    Ok(())
}
```

Note the ordering: status flip + outcome stored under the lock; abort + sink fire outside. async fn complete_internal(&self, job_id: JobId, outcome: JobOutcome) {
```rust
    let sink = {
        let mut jobs = self.jobs.lock();

            tracing::warn!(%job_id, "complete_internal for unknown job; dropping");
            return;
        };
        // Idempotent: if already terminal (e.g., cancel raced ahead),
        // don't overwrite status/outcome.
        if matches!(
            job.status,
            JobStatus::Completed | JobStatus::Failed | JobStatus::Cancelled
        ) {
            return;
        }
        job.status = match &outcome {
            JobOutcome::Success { .. } => JobStatus::Completed,
            JobOutcome::Failed { .. } => JobStatus::Failed,
            JobOutcome::Cancelled => JobStatus::Cancelled,
            _ => JobStatus::Failed,

        job.abort_handle = None;
```

```rust
        args: serde_json::Value,
        ctx: ExecCtx,
    ) -> Result<InvokeOutcome, ToolError> {

            .map_err(|e| ToolError::InvalidArgs(e.to_string()))?;

        let deadline = ctx
            .deadline
            .map(|d| d.saturating_duration_since(std::time::Instant::now()))
            .unwrap_or_else(|| Duration::from_secs(DEFAULT_DEADLINE_SECS));
        let cancel = ctx.cancel.clone();

        });

async fn run_cargo_nextest(
    args: RunTestsArgs,
    deadline: Duration,
    cancel: tokio_util::sync::CancellationToken,
) -> JobOutcome {
    let mut cmd = Command::new("cargo");
    cmd.arg("nextest").arg("run");
    if let Some(pkg) = &args.package {
        cmd.arg("-p").arg(pkg);
    }
    if let Some(filter) = &args.filter {
        cmd.arg(filter);
    }
    cmd.stdout(Stdio::piped()).stderr(Stdio::piped());

    let mut child = match cmd.spawn() {
        Ok(c) => c,
        Err(e) => {
            return JobOutcome::Failed {
                message: format!("failed to spawn cargo nextest: {e}"),
            };
        }
    };

    let wait = child.wait_with_output();
    tokio::select! {
        result = wait => match result {
            Ok(output) => {
                let stdout = String::from_utf8_lossy(&output.stdout);
                let stderr = String::from_utf8_lossy(&output.stderr);
                JobOutcome::Success {
                    result: ToolResult::Output {
                        value: serde_json::json!({
                            "stdout": truncate(&stdout),
                            "stderr": truncate(&stderr),
                            "exit_code": output.status.code(),
                        }),
                    },
                }
            }
            Err(e) => JobOutcome::Failed {
                message: format!("cargo nextest wait failed: {e}"),
            },
        },
        () = cancel.cancelled() => {
            // child.kill() requires a &mut Child. We dropped wait_with_output's
            // future, which consumed `child` — restructure: keep child mutable
            // outside wait. See alternative below.
            JobOutcome::Cancelled
        }
        () = tokio::time::sleep(deadline) => {
            JobOutcome::Failed { message: "timeout".into() }
        }
    }
}

fn truncate(s: &str) -> String {
    let bytes = s.as_bytes();
    if bytes.len() <= TRUNCATE_HEAD_BYTES + TRUNCATE_TAIL_BYTES {
        return s.to_string();
    }
    let head = std::str::from_utf8(&bytes[..TRUNCATE_HEAD_BYTES])
        .unwrap_or("");
    let tail_start = bytes.len() - TRUNCATE_TAIL_BYTES;
    let tail = std::str::from_utf8(&bytes[tail_start..]).unwrap_or("");
    let elided = bytes.len() - TRUNCATE_HEAD_BYTES - TRUNCATE_TAIL_BYTES;
    format!("{head}\n... [{elided} bytes elided] ...\n{tail}")
}
```
